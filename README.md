# Political Market Analysis Terminal

A Next.js application for analyzing political trends through market-style visualizations.

## Features

- Market Overview with sector analysis
- Stress Testing indicators
- Political Commodities tracking
- Historical trajectory visualization
- Mathematical framework display

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

## Deploy to Vercel

1. Push to GitHub
2. Import project in Vercel
3. Deploy with default settings

## Technologies

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
- Recharts
